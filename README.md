# Slack AI Agent

A Slack AI Agent built with Node.js, OpenAI, Slack Bolt, Express, and PostgreSQL.

## Features

- Detects new Slack members
- Analyzes member profiles using OpenAI
- Stores analysis in PostgreSQL
- Posts analysis results to a private Slack channel
- Health check endpoint

## Installation

```bash
npm install
```

## Run

```bash
node index.js
```

## Environment Variables

Create a `.env` file and add your Slack, OpenAI, and database credentials.
